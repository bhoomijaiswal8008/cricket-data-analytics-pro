# 🏏 Cricket Stats Analytics: From Raw Data to Winning Insights! 🚀

Welcome! If you love cricket and data, you are in the right place. I took a massive dataset of ODI Cricket and performed some "Excel Sorcery" to turn boring rows into interactive charts and dashboards. 

Why? Because even data deserves a stadium-like atmosphere! 🏟️✨

### 🧐 What’s the Magic?
I analyzed legendary players like **Virat Kohli**, **KC Sangakkara**, and **RG Sharma** to see who truly dominates the field.
- **Data Cleaning:** Fixed the mess (just like a clean sweep!).
- **Pivot Tables:** Summarized 1000s of rows into meaningful insights faster than a Dhoni stumping. ⚡
- **Interactive Dashboards:** Created visual representations of matches won, player awards, and strike rates.

### 🛠️ Prerequisites (What you need)
To run or view this project, you'll need:
1. **Microsoft Excel (2019+ recommended):** For viewing the Pivot Tables and Slicers. 💻
2. **Basic Cricket Knowledge:** If you know what a 'Six' or a 'Duck' is, you're good to go! 🦆
3. **Curiosity:** To explore how different teams perform under pressure.

### 📂 Repository Structure
- `CRICKET.xlsx`: The main powerhouse containing raw data, pivots, and charts.
- `screenshots/`: Quick visual previews for those in a hurry. 🖼️
